---
title: Cannot Create Project
description: Fix project creation errors in SIVAKUMAR S.
---

## Symptoms
- "Create Project" button is unresponsive or shows error
- Project not appearing in dashboard

## Possible Causes
- Missing required fields
- Browser issues

## Quick Fix
<Steps>
  <Step title="Check required fields">
    Ensure you entered both project name and description.
  </Step>
  <Step title="Clear browser cache">
    Log out, clear your browser cache, and log back in.
  </Step>
</Steps>
