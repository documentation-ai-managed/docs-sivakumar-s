## Overview

To start using SIVAKUMAR S, create your first documentation project from the dashboard.

## Steps

1. Login to your SIVAKUMAR S account
2. Click "New Project"
3. Name your project and add a description
4. Click "Create"

Your project is now ready. Add pages and organize your docs in the left sidebar.
