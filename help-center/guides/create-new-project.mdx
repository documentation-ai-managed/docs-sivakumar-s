---
title: Creating a New Documentation Project
description: Start your first doc project with SIVAKUMAR S.
---

## Steps

<Steps>
  <Step title="Go to Dashboard">
    Navigate to your dashboard after login at https://dashboard.sivakumars.com.
  </Step>
  <Step title="Click New Project">
    Press the "New Project" button in the top right corner.
  </Step>
  <Step title="Enter details">
    Fill in the project name and (optionally) a description.
  </Step>
  <Step title="Create">
    Click "Create". Your project is now listed in the dashboard and ready for documentation.
  </Step>
</Steps>
